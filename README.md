## 使用方法

1. 将你想要的 skill 文件夹复制到 Claude Code 的 skills 目录：
   - **Windows**: `C:\Users\你的用户名\.claude\skills\`
   - **macOS**: `~/.claude/skills/`
2. 按照每个 skill 内的 README.md 完成配置
3. 在 Claude Code 中直接用自然语言使用
